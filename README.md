# ken7y's Homebrew Tap

This is a [Homebrew tap](https://docs.brew.sh/Taps) for my macOS applications.

## Installation

```bash
brew tap ken7y/tap
```

## Available Casks

### [WiFiBuddy](https://github.com/ken7y/MacWifiBuddy)

WiFi diagnostics and network monitoring for macOS menu bar.

```bash
brew install wifibuddy
```

## Uninstall

```bash
brew untap ken7y/tap
```
